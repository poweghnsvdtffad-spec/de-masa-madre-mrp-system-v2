# De Masa Madre - MRP System

Lightweight Material Requirements Planning (MRP) system for food production businesses.

## Tech Stack
- React 19
- Vite
- Framer Motion
- Lucide Icons
- Vanilla CSS

## Features
- **Dashboard**: Real-time KPIs for inventory turnover, waste rate, and schedule adherence.
- **Orders**: Manage pending and production orders.
- **Production**: Automatic calculation of component requirements based on recipes.
- **Inventory**: Stock management with safety stock alerts.
- **AI Assistant**: Conversational agent for stock queries and integration guides.

## Setup
1. Clone the repo
2. `npm install`
3. `npm run dev`
